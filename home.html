Introduccion

Esta es la documentación no oficial del lenguaje C++. Aquí encontraras algunos conceptos básicos los cuales necesitaras para adentrarte al mismo.
NOTA: Recomendable utilizar el IDE DEV C++ para el desarrollo del lenguaje.

Variables

Las variables son contenedores para almacenar valores de datos.
En C ++, existen diferentes tipos de variables (definidas con diferentes palabras clave), por ejemplo:

  • int - almacena enteros (números enteros), sin decimales, como 123 o -123.
  • double - almacena números de punto flotante, con decimales, como 19,99 o -19,99.
  • char - almacena caracteres individuales, como 'a' o 'B'. Los valores de caracteres están rodeados por comillas simples.
  • string - almacena texto, como "Hola mundo". Los valores de cadena están rodeados por comillas dobles.
  • bool - almacena valores con dos estados: verdadero o falso.

Declarar variables

La sintaxis para declarar las variables en C++ es la siguiente:
type variable_name = value;

Hello world
Es realmente sencillo construir el famoso "Hello World" en este lenguaje, veamos como hacerlo:
#include<iostream>;
using namespace std;
main() {
 cout << "Hello world";
}
Arrays

Las matrices se utilizan para almacenar varios valores en una sola variable, en lugar de declarar variables independientes para cada valor.
Para declarar una matriz, defina el tipo de variable, especifique el nombre de la matriz seguido de corchetes y especifique la cantidad de elementos que debe almacenar:

string cars[5];

Ahora hemos declarado una variable que contiene una matriz de cuatro cadenas. Para insertar valores en él, podemos usar un literal de matriz: coloque los valores en una lista separada por comas, entre llaves:

string cars[5] = {"Ferrari", "Lamborghini", "Ford", "Tesla"} ;

Ciclos condicionales

C ++ admite las condiciones lógicas habituales de las matemáticas:

  • a < b menor que.
  • a <= b menor o igual que.
  • a > b mayor que.
  • a >= b mayor o igual que.
  • a == b igual a.
  • a != b diferente de.

Puede utilizar estas condiciones para realizar diferentes acciones para diferentes decisiones. C ++ tiene las siguientes declaraciones condicionales:

  • Se usa if para especificar un bloque de código que se ejecutará, si una condición especificada es verdadera.
  • Se usa else para especificar un bloque de código que se ejecutará, si la misma condición es falsa.
  • Se usa else if para especificar una nueva condición para probar, si la primera condición es falsa.
  • Se usa switch para especificar muchos bloques alternativos de código a ejecutar.

En el siguiente ejemplo, probamos dos valores para averiguar si 20 es mayor que 18. Si la condición es true, imprima algo de texto:

if ( 20 > 18 ) {
  cout >> "20 si es mayor que 18";
}

NOTA: Importante usar if en minusculas. De lo contrario podría generar error.
Ciclos repetitivos

Los bucles pueden ejecutar un bloque de código siempre que se alcance una condición específica.
Los bucles son útiles porque ahorran tiempo, reducen errores y hacen que el código sea más legible.

Ciclo while

El bucle while recorre un bloque de código siempre que una condición especificada sea true.
En el siguiente ejemplo, el código del bucle se ejecutará una y otra vez siempre que una variable (i) sea menor que 5:

int i = 0;
while ( i < 5 ) {
  cout << i << "\n";
  i++;
}

NOTA: Si la variable i no es aumentada, el ciclo sera infinito.

Ciclo for

Cuando sepa exactamente cuántas veces desea recorrer un bloque de código, use el ciclo for en lugar del ciclo while:

for ( statement 1; statement 2; statement 3 ) {
  //code block to be executed
}

La declaración 1 se ejecuta (una vez) antes de la ejecución del bloque de código.
La declaración 2 define la condición para ejecutar el bloque de código.
La declaración 3 se ejecuta (cada vez) después de que se haya ejecutado el bloque de código.
El siguiente ejemplo imprimirá los números del 0 al 4:

for ( int i = 0; i < 4; i++ ) {
  cout << i << "\n";
}
